## About
A service and series of tools to search the comprehensive rules (CR) of Magic: the Gathering.

## How to Use
For the vast majority of users that just want to search through the CR, go [here](https://frantic-search.cr) to search through the latest edition of the CR.

If your interest lays outside of the web UI, there is a CLI tool to do the searching via the command line. That can be built and installed via Rust's build system, cargo.
